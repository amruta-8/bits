SESSION 1

Linear Systems & Solving \(Ax=b\)

Three ways to read \(Ax=b\)

Row view: each equation is a hyperplane; the solution is their intersection.
Column view: \(b\) is a linear combination of the columns of \(A\) with weights \(x\). So \(Ax=b\) is solvable iff \(b\in\) column space of \(A\).
Transformation view: \(A\) is a function mapping \(x\mapsto Ax\); solving asks "which input maps to \(b\)?".

Gaussian elimination → Row Echelon Form (REF)

Use the three elementary row operations (swap rows; scale a row by a non-zero constant; add a multiple of one row to another) to create zeros below each pivot (the first non-zero entry of a row). The result is REF (staircase of zeros below pivots); reducing further to leading 1s with zeros above too gives RREF.

What REF gives you instantly Rank = number of non-zero rows = number of pivots.
Determinant (square \(A\)) = product of the pivots, times \((-1)^{\#\text{row swaps}}\).
Pivot columns ↔ basis of the column space; free (non-pivot) columns ↔ null-space directions.

Existence & uniqueness of solutions

Condition (on augmented \([A|b]\))Solutions
rank\(A\) < rank\([A|b]\) (a row \(0=\)nonzero appears)No solution (inconsistent)
rank\(A\) = rank\([A|b]\) = number of unknownsUnique solution
rank\(A\) = rank\([A|b]\) < number of unknownsInfinitely many (free variables)
Structure of the general solution When there are free variables: general solution = one particular solution + (any combination of null-space vectors): $$x = x_p + \sum_k \lambda_k\, n_k,\qquad A x_p=b,\ A n_k=0.$$ Get \(x_p\) by setting all free variables to 0; get each \(n_k\) by setting one free variable to 1 and the rest to 0 in the homogeneous system \(Ax=0\).

Determinant, inverse, rank — the toolbox

  • Determinant via cofactor expansion along any row/column: \(\det A=\sum_j (-1)^{i+j}a_{ij}M_{ij}\). Properties: \(\det(AB)=\det A\det B\), \(\det(A^\top)=\det A\), \(\det(A^{-1})=1/\det A\), \(\det(cA)=c^n\det A\).
  • Inverse exists ⇔ \(\det A\neq0\) ⇔ rank \(=n\) ⇔ columns independent ⇔ \(Ax=0\) only at \(x=0\). For 2×2 use the swap-negate formula; for larger, Gauss–Jordan on \([A\,|\,I]\to[I\,|\,A^{-1}]\), then \(x=A^{-1}b\).
  • Rank–nullity: \(\operatorname{rank}(A)+\dim(\text{null space})=\) number of columns. Memorise this — it answers many "dimension of null space" parts in one line.
Likely exam asks (Q1) "Find the echelon form of \(A\), hence its determinant and rank" · "Solve \(Ax=b\) / give all solutions" · "What can you conclude about \(Ax=b\) for this \(b\)? (none/unique/infinite)" · "Find \(A^{-1}\) and use it to solve \(Ax=b\)".
SESSION 2

Vector Spaces, Independence, Basis & Null Space

Vector space & subspace

A vector space is a set closed under vector addition and scalar multiplication satisfying the usual 8 axioms (associativity, commutativity, zero vector, inverses, distributivity, …). A subspace \(W\) of a vector space is a non-empty subset that is itself a vector space.

Subspace test (3 conditions) \(W\) is a subspace ⇔ (1) \(\mathbf 0\in W\); (2) closed under addition (\(u,v\in W\Rightarrow u+v\in W\)); (3) closed under scalar multiplication (\(u\in W,\ c\in\mathbb R\Rightarrow cu\in W\)).
Lines/planes through the origin are subspaces; anything not containing \(\mathbf 0\) (e.g. \(x=1\), or \(\{x\ge0\}\)) is not.

Span is always a subspace: \(\text{span}\{v_1,\dots,v_k\}=\{\sum c_iv_i\}\) automatically satisfies all three tests — a frequently-used fact.

Linear independence

Definition & test \(v_1,\dots,v_k\) are linearly independent if \(\;c_1v_1+\dots+c_kv_k=0\;\) forces \(c_1=\dots=c_k=0\). Otherwise dependent (one is a combination of the others).
Test: put the vectors as columns of \(A\), row-reduce. Independent ⇔ every column has a pivot ⇔ (if square) \(\det A\neq0\). A non-pivot column gives an explicit dependency relation.

Basis & dimension

A basis is a linearly-independent set that spans the space; dimension = number of vectors in any basis. In \(\mathbb R^n\), any \(n\) independent vectors form a basis. Key subspaces of a matrix \(A\):

Null space (kernel)\(\{x:Ax=0\}\)one vector per free variablenullity = #cols − rank

SESSION 3

Analytic Geometry — Inner Products, Norms, Orthogonality

Inner products

The dot product \(x\cdot y=x^\top y\) generalises to \(\langle x,y\rangle = x^\top A y\) for a fixed matrix \(A\). An inner product must be (i) symmetric \(\langle x,y\rangle=\langle y,x\rangle\), (ii) bilinear, and (iii) positive definite \(\langle x,x\rangle>0\) for \(x\neq0\).

When does \(x^\top A y\) define an inner product? Exactly when \(A\) is symmetric AND positive definite (SPD). Three equivalent PD tests:
• Sylvester: all leading principal minors \(>0\) (e.g. \(2\times2\): \(a_{11}>0\) and \(\det A>0\)).
• all eigenvalues \(>0\).
• \(x^\top A x>0\) for all \(x\neq0\) — often shown by writing it as a sum of squares (e.g. \(3x_1^2+x_2^2-2x_1x_2=2x_1^2+(x_1-x_2)^2>0\)).

Norm, distance, angle, orthogonality

\(\displaystyle \|x\|=\sqrt{\langle x,x\rangle}=\sqrt{x^\top A x}\) · distance \(d(x,y)=\|x-y\|=\sqrt{(x-y)^\top A (x-y)}\)
angle \(\displaystyle \cos\theta=\frac{\langle x,y\rangle}{\|x\|\,\|y\|}\) · \(x\perp y\) (orthogonal) ⇔ \(\langle x,y\rangle=0\)
Cauchy–Schwarz: \(|\langle x,y\rangle|\le\|x\|\,\|y\|\).

Exam tip: with a non-standard inner product \(\langle x,y\rangle=x^\top A y\), always compute \(Ay\) first, then dot with \(x\). For distance, form \(w=x-y\) and evaluate \(w^\top A w\).

Orthonormal bases & Gram–Schmidt

An orthonormal set has \(\langle e_i,e_j\rangle=\delta_{ij}\) (mutually orthogonal, unit norm). Gram–Schmidt turns any independent set \(\{v_1,v_2,\dots\}\) into an orthogonal one, then normalise:

$$u_1=v_1,\quad u_2=v_2-\frac{\langle v_2,u_1\rangle}{\langle u_1,u_1\rangle}u_1,\quad u_3=v_3-\frac{\langle v_3,u_1\rangle}{\langle u_1,u_1\rangle}u_1-\frac{\langle v_3,u_2\rangle}{\langle u_2,u_2\rangle}u_2,\ \dots$$ then \(e_i=u_i/\|u_i\|\). (Use whichever inner product the question specifies, standard or \(A\)-weighted.)

An orthogonal matrix \(Q\) (columns orthonormal) satisfies \(Q^\top Q=I\), so \(Q^{-1}=Q^\top\); it preserves lengths and angles.

Likely exam asks (Q4) "Prove \(A\) is symmetric PD, hence defines an inner product" · "Using this inner product, find the distance between… / the angle between… / whether these are orthogonal" · "Find the norm of \(v\)" · "Apply Gram–Schmidt to \(v_1,v_2\)".
SESSION 4

Matrix Decomposition I — Determinant, Eigenvalues, Cholesky

Eigenvalues & eigenvectors

Definition & computation \(Av=\lambda v\) (\(v\neq0\)). Eigenvalues solve the characteristic equation \(\det(A-\lambda I)=0\). For each \(\lambda\), eigenvectors span the null space of \((A-\lambda I)\) (the eigenspace).
Shortcuts: \(\sum_i\lambda_i=\operatorname{tr}(A)\); \(\prod_i\lambda_i=\det(A)\); a triangular matrix's eigenvalues are its diagonal entries; eigenvalues of \(A^k\) are \(\lambda^k\) (same eigenvectors); \(A\) and \(A^\top\) share eigenvalues.

Diagonalizability

When is \(A\) diagonalizable (\(A=PDP^{-1}\))? For every eigenvalue, the geometric multiplicity (\(=\dim\) eigenspace \(=n-\text{rank}(A-\lambda I)\)) must equal its algebraic multiplicity (its repetition in the char. polynomial).
• \(n\) distinct eigenvalues ⇒ always diagonalizable (distinct eigenvalues give independent eigenvectors).
• Symmetric ⇒ orthogonally diagonalizable (Spectral Theorem: real eigenvalues, orthonormal eigenvectors, \(A=Q\Lambda Q^\top\)).
• A repeated eigenvalue with too few independent eigenvectors (e.g. \(\begin{bmatrix}1&1\\-1&-1\end{bmatrix}\), or any non-zero nilpotent) is NOT diagonalizable.

Eigendecomposition: \(A=P D P^{-1}\) with eigenvectors as columns of \(P\) and eigenvalues on the diagonal of \(D\). For symmetric \(A\), \(P\) is orthogonal so \(A=Q\Lambda Q^\top\). Verify by checking \(AP=PD\).

Cholesky decomposition

Every SPD matrix factors uniquely as \(A=LL^\top\) with \(L\) lower-triangular, positive diagonal:

$$l_{jj}=\sqrt{\,a_{jj}-\textstyle\sum_{kj).$$ Use: solve \(Ax=b\) via \(Ly=b\) (forward) then \(L^\top x=y\) (back) — ~2× faster than LU; also for sampling correlated data. A successful Cholesky is itself a proof that \(A\) is SPD.
Likely exam asks (Q3/Q4) "Find eigenvalues & eigenvectors" · "Is \(A\) diagonalizable? Justify" · "Find \(P,D\) with \(AP=PD\) (or \(A=PDP^{-1}\))" · "Why is \(A^\top A\) always diagonalizable?" (it's symmetric) · "Find the Cholesky factor \(L\)" · "Trace/det of \(A^k\) via eigenvalues".
SESSION 5

Matrix Decomposition II — Eigendecomposition & SVD

Singular Value Decomposition

Every \(m\times n\) matrix factors as \(A=U\Sigma V^\top\): \(U\) (\(m\times m\), orthonormal left-singular vectors), \(\Sigma\) (\(m\times n\), non-negative singular values on the diagonal, descending), \(V\) (\(n\times n\), orthonormal right-singular vectors). Unlike eigendecomposition, SVD exists for any matrix (rectangular, non-square).

SVD recipe (the exam method) 1. Form \(A^\top A\) (\(n\times n\), symmetric PSD). Its eigenvalues are \(\sigma_i^2\); its orthonormal eigenvectors are the columns of \(V\).
2. Singular values \(\sigma_i=\sqrt{\lambda_i}\) (descending) fill \(\Sigma\).
3. Left vectors \(u_i=\dfrac{1}{\sigma_i}A v_i\) for \(\sigma_i>0\); complete \(U\) to an orthonormal basis for the zero singular values.
4. Verify \(A=U\Sigma V^\top\). (If \(m<n\), it's usually easier to start from \(AA^\top\) instead.)
Things SVD hands you Spectral norm \(\|A\|_2=\sigma_{\max}\). Frobenius norm \(\|A\|_F^2=\sum_i\sigma_i^2=\operatorname{tr}(A^\top A)\). Rank = number of non-zero singular values. Best rank-\(k\) approximation (Eckart–Young) = \(\sum_{i=1}^k\sigma_iu_iv_i^\top\); the best rank-1 is \(\sigma_1u_1v_1^\top\).

For a symmetric PD matrix the SVD and eigendecomposition coincide (\(\sigma_i=\lambda_i\), \(U=V=Q\)). For a general symmetric matrix \(\sigma_i=|\lambda_i|\).

Likely exam asks (Q3) "Find the SVD \(A=U\Sigma V^\top\) of this 2×2 / 3×2 matrix" · "Find \(\Sigma\) and \(V\) (no need for \(U\))" · "Find the left/right singular vectors and the spectral norm" · "Best rank-1 approximation".
SESSION 6

Vector Calculus I — Derivatives, Gradients, Jacobian

Gradients

For \(f:\mathbb R^n\to\mathbb R\), the gradient stacks the partial derivatives: \(\nabla f=\big[\frac{\partial f}{\partial x_1},\dots,\frac{\partial f}{\partial x_n}\big]^\top\). It points in the direction of steepest increase, and \(\nabla f=0\) marks critical points.

Worked pattern For \(f(x,y)=x^2+xy+2y^2-4x+3\): \(\dfrac{\partial f}{\partial x}=2x+y-4\), \(\dfrac{\partial f}{\partial y}=x+4y\) ⇒ \(\nabla f=[\,2x+y-4,\ x+4y\,]^\top\). Evaluate at a point by substitution.

Jacobian (vector-valued functions)

For \(f:\mathbb R^n\to\mathbb R^m\), the Jacobian \(J\) is the \(m\times n\) matrix \(J_{ij}=\partial f_i/\partial x_j\) — one gradient (as a row) per output component.

\(f(x)=\begin{bmatrix}x_1^2x_2\\5x_1+\sin x_2\end{bmatrix}\Rightarrow J=\begin{bmatrix}2x_1x_2 & x_1^2\\ 5 & \cos x_2\end{bmatrix}\).   Linear map \(f(x)=Wx\Rightarrow J=W\).

Gradient identities to memorise

\(\nabla(b^\top x)=b\) · \(\nabla(x^\top x)=2x\) · \(\nabla(x^\top Ax)=(A+A^\top)x=2Ax\) (if \(A\) symmetric) · \(\nabla\|Ax-b\|^2=2A^\top(Ax-b)\).
Matrix derivatives: \(\frac{\partial}{\partial X}\sum X_{ij}^2=2X\); \(\frac{\partial}{\partial X}\det X = \det(X)\,X^{-\top}\) (2×2: the adjugate).
Likely exam asks (Q5) "Find \(\nabla f\) (and evaluate at a point)" · "Is vector \(v\) orthogonal to \(\nabla f\) here?" · "Find the Jacobian of this vector function" · "Find the gradient of \(\|Ax-b\|^2\) / \(x^\top Q x\)".
SESSION 7

Vector Calculus II — Chain Rule, Backpropagation, Autodiff

The multivariate chain rule

Core formula If \(f\) depends on \(x_1,\dots,x_n\) which each depend on \(t\): \(\;\dfrac{df}{dt}=\sum_i\dfrac{\partial f}{\partial x_i}\dfrac{dx_i}{dt}\).
For several parameters \(r,s\): \(\dfrac{\partial f}{\partial r}=\sum_i\dfrac{\partial f}{\partial x_i}\dfrac{\partial x_i}{\partial r}\) (one term per intermediate variable / per path).

Worked: \(f(x,y,z)=x^2+y^2+z^2+3xyz\), \(x=t,y=t^2,z=t^3\): \(\frac{df}{dt}=(2x+3yz)(1)+(2y+3xz)(2t)+(2z+3xy)(3t^2)=24t^5+4t^3+2t\).

Computation graphs & backpropagation

Backpropagation is just the chain rule applied on a computation graph, evaluated backwards. Forward pass computes each node; backward pass multiplies local derivatives from output to input, summing over all paths into a node: \(\bar u=\sum_i \bar v_i\,\frac{\partial v_i}{\partial u}\).

Two derivatives that always show up Sigmoid \(q=\sigma(p)=\frac{1}{1+e^{-p}}\Rightarrow \dfrac{\partial q}{\partial p}=q(1-q)\).
Tanh \(s=\tanh(r)\Rightarrow \dfrac{\partial s}{\partial r}=1-s^2\).
These let you write \(\partial L/\partial w\) for a small neuron graph in closed form.

Linearization & automatic differentiation

Linearization (1st-order Taylor): \(L(x)\approx f(x_0)+\nabla f(x_0)^\top(x-x_0)\) — the local linear approximation. Automatic differentiation evaluates exact derivatives by composing local rules along the graph (forward or reverse mode); reverse mode = backprop.

Likely exam asks (Q6) "Using the chain rule find \(df/dt\) (or \(\partial f/\partial r,\partial f/\partial s\))" · "Draw the computation graph and find \(\partial L/\partial w\)" · "Show \(\partial \sigma/\partial p=\sigma(1-\sigma)\)" · "Find the gradient of a matrix-valued \(f(A)=A^\top A\)" · "Linearize \(f\) about a point".
SESSION 8

Vector Calculus III — Hessian, Taylor, Optimization

Higher-order derivatives & the Hessian

The Hessian is the symmetric matrix of second partials \(H=\begin{bmatrix}f_{xx}&f_{xy}\\f_{yx}&f_{yy}\end{bmatrix}\) (\(f_{xy}=f_{yx}\) by Clairaut's theorem). It captures local curvature.

Critical points & the second-derivative test

Classify a critical point (where \(\nabla f=0\)) 2-variable test with \(D=f_{xx}f_{yy}-f_{xy}^2=\det H\):
• \(D>0\) and \(f_{xx}>0\) → local minimum
• \(D>0\) and \(f_{xx}<0\) → local maximum
• \(D<0\) → saddle point
• \(D=0\) → test inconclusive
General: Hessian positive definite → min; negative definite → max; indefinite → saddle. (Check eigenvalue signs.)

Worked: \(f=x^2+xy+2y^2-4x+3\). \(\nabla f=0\Rightarrow(16/7,-4/7)\). \(H=\begin{bmatrix}2&1\\1&4\end{bmatrix}\), \(\det H=7>0\), \(f_{xx}=2>0\) ⇒ local minimum.

Taylor series

Univariate Taylor about \(a\) $$f(x)=f(a)+f'(a)(x-a)+\frac{f''(a)}{2!}(x-a)^2+\frac{f'''(a)}{3!}(x-a)^3+\cdots$$ Maclaurin (\(a=0\)) staples: \(e^x=1+x+\frac{x^2}{2}+\dots\); \(\cos x=1-\frac{x^2}{2!}+\frac{x^4}{4!}-\dots\); \(\sin x=x-\frac{x^3}{3!}+\dots\). Multivariate (2nd order): \(f(x)\approx f(a)+\nabla f(a)^\top\Delta+\tfrac12\Delta^\top H\Delta\).

Trick for composites like \(e^{\sin(x^2)}\): substitute the small-\(x\) expansions (\(\sin(x^2)\approx x^2\), \(e^t\approx1+t+\dots\)) and keep terms up to the needed degree → \(1+x^2\) to degree 2.

Convexity

\(f\) is convex iff its Hessian is positive semi-definite everywhere (equivalently the chord lies above the graph). Convexity guarantees any critical point is a global minimum — the reason squared-error/​log-loss are nice to optimise. Norms are convex (triangle inequality); \(g(Ax+b)\) is convex if \(g\) is.

Likely exam asks (Q5/Q6) "Find the critical point(s)" · "Classify as min/max/saddle using the Hessian" · "Find the Taylor polynomial of degree \(n\) about \(x=a\)" · "Just by observing \(f\), give its smallest/largest value" · "Show \(J\) is convex (Hessian PD)" · "Maximize a profit/utility function via Hessian definiteness".

EG1. Is \(A\) diagonalizable? — Dec 2025 Regular, Q3b · 2M
Question (verbatim) Is the matrix \(A=\begin{bmatrix}1&1\\-1&-1\end{bmatrix}\) diagonalizable? Justify your answer. [2M]

Trace \(=0\), \(\det=0\) ⇒ characteristic equation \(\lambda^2=0\) ⇒ \(\lambda=0\) with algebraic multiplicity 2. But rank\((A)=1\), so geometric multiplicity \(=n-\text{rank}=2-1=1\). Question (verbatim) A data scientist is told the characteristic equation of a matrix has all roots distinct and real. He assumes (a) the matrix is bound to be invertible, and (b) also diagonalizable. Is he justified? Give reasons. [5M]
Solution

(a) Not justified. Distinct real eigenvalues do not prevent one of them being zero. A zero eigenvalue means \(\det A=\prod\lambda_i=0\), so \(A\) would be singular (non-invertible). Invertibility needs all eigenvalues \(\neq0\), which "distinct & real" does not guarantee.

(b) Justified. \(n\) distinct eigenvalues give \(n\) eigenvectors that are automatically linearly independent, so the eigenvector matrix \(P\) is invertible and \(A=PDP^{-1}\) — i.e. \(A\) is diagonalizable.

ES5. Inner-product matrix with parameter \(\rho\); orthogonal complement — 2022 EndSem Regular, Q3a · 6M · L3

ES6. Eigenvalues/vectors; diagonalize; singular values — Feb 2026 Regular EndSem, Q1 · 8M · L4/L5
Question (verbatim) \(A=\begin{bmatrix}3&1&0\\1&3&0\\0&0&4\end{bmatrix}\). (a) Eigenvalues and eigenvectors. (b) \(D,P\) with \(AP=PD\). (c) Singular values of \(B=\begin{bmatrix}\sqrt3&1/\sqrt3&0\\0&\sqrt{8/3}&0\\0&0&2\end{bmatrix}\). [5+2+1M]

(b) \(P=\begin{bmatrix}1&1&0\\-1&1&0\\0&0&1\end{bmatrix},D=\text{diag}(2,4,4)\).

(c) \(B^\top B=A\Rightarrow\) singular values \(=\sqrt{\text{eig}(A)}=\sqrt2,2,2\).

ES7. Trace & determinant of matrix powers via eigenvalues — 2023 EndSem Regular, Q1B · 3M · L4
Question (verbatim) \(C=\begin{bmatrix}6&-33&19\\6&-8&34\\9&-33&16\end{bmatrix}\). (i) Trace of \(C^6\). (ii) Determinant of \(C^7\). [3M]
Solution

Method (the examinable idea): the eigenvalues of \(C^k\) are \(\lambda_i^k\) (same eigenvectors), so \(\operatorname{tr}(C^k)=\sum_i\lambda_i^k\) and \(\det(C^k)=(\det C)^k=\big(\prod_i\lambda_i\big)^k\). You never form \(C^6\) or \(C^7\) — just the eigenvalues.

With the answer-key eigenvalues \(\lambda=14,3,-3\): (i) \(\operatorname{tr}(C^6)=14^6+3^6+(-3)^6\). (ii) \(\det(C^7)=(\det C)^7=(14\cdot3\cdot(-3))^7=(-126)^7\).

Trust the method above and the official key's eigenvalues \(\{14,3,-3\}\); re-check the exact entries against your own copy before relying on the numbers.
ES8. Frobenius-norm SVD identities — 2022 EndSem Regular, Q2b · 4M · L5
Question (verbatim) \(A\in\mathbb R^{n\times n}\), \(A=U\Sigma V^\top\), \(\|\Sigma\|_F^2=\gamma\). \(B=A^\top A,\alpha=\sum_iB_{ii}\). (a) \(\alpha=\gamma\)? (b) \(\alpha=\sqrt\gamma\)? (c) For \(C=AA^\top,\beta=\sum_iC_{ii}\): \(\beta=\gamma^2\)? [4M]
Solution

\(\gamma=\sum\sigma_i^2\). (a) \(\alpha=\operatorname{tr}(A^\top A)=\operatorname{tr}(\Sigma^\top\Sigma)=\sum\sigma_i^2=\gamma\) ⇒ TRUE. (b) value is \(\gamma\), not \(\sqrt\gamma\) ⇒ FALSE. (c) \(\beta=\operatorname{tr}(AA^\top)=\gamma\neq\gamma^2\) ⇒ FALSE.

ES9. Gradient & linear approximation of a vector function — 2022 EndSem Regular, Q3b · 3M · L6

ES11. Critical points of \((x^2+y^2)e^{-(x^2+y^2)}\) — 2023 EndSem Regular, Q2C · 3.5M · L8
Question (verbatim) \(f(x,y)=(x^2+y^2)e^{-(x^2+y^2)}\). (i) Find all critical points. (ii) Is \((0,0)\) a minimum? [1.5+2M]
Solution

(i) \(f_x=2x\,e^{-r^2}(1-r^2)=0\) (\(r^2=x^2+y^2\)) ⇒ \(x=y=0\) or \(r^2=1\). Critical set \(=\{(0,0)\}\cup\{x^2+y^2=1\}\).

(ii) \(H(0,0)=\begin{bmatrix}2&0\\0&2\end{bmatrix}\) PD ⇒ local minimum (value 0, and \(f\ge0\)).

ES12. Hessian ⇒ convexity of a least-squares loss — Feb 2026 Regular EndSem, Q4a,b · 4M · L8
Question (verbatim) \(\hat y_i=w_0+w_1x_i\), data \((5,3),(6,5)\), \(J=\tfrac12\sum_{i=1}^2(\hat y_i-y_i)^2\). (a) Write \(J\), compute the Hessian, prove convexity. (b) Compute \(\nabla J\). [3+1M]
Solution

(a) \(J=\tfrac12[(w_0+5w_1-3)^2+(w_0+6w_1-5)^2]\). \(H=\begin{bmatrix}2&11\\11&61\end{bmatrix}\); \(2>0,\det H=1>0\) ⇒ PD ⇒ convex.

ES13. Convexity of norms and affine compositions — 2022 EndSem Regular, Q2a · 4M · L8
Question (verbatim) (i) Prove \(\|x\|_1\) is convex. (ii) If \(g\) convex, prove \(h(x)=g(Ax+b)\) is convex. [4M]
Solution

(i) \(\|\lambda x+(1-\lambda)y\|_1\le\lambda\|x\|_1+(1-\lambda)\|y\|_1\) (triangle inequality + homogeneity) = convexity.
(ii) \(h(\lambda x+(1-\lambda)y)=g(\lambda(Ax+b)+(1-\lambda)(Ay+b))\le\lambda h(x)+(1-\lambda)h(y)\) ⇒ convex.

🎯 Last-Minute Exam-Hall Card

  1. Always row-reduce cleanly and keep the augmented bar — rank, det, solvability and null space all fall out of one echelon form.
  2. Determinant sign: each row swap multiplies \(\det\) by \(-1\); det = product of pivots otherwise.
  3. "Prove it's an inner product" = show \(A=A^\top\) + positive definite (minors \(>0\) or sum of squares). This template recurs every paper.
  4. "Prove subspace" = \(0\in W\), closed under \(+\) and scalar \(\cdot\). For null spaces this is automatic from linearity.
  5. Weighted inner product \(\langle x,y\rangle=x^\top Ay\): compute \(Ay\) first, then dot with \(x\). For distance use \(w=x-y\), evaluate \(w^\top Aw\).
  6. SVD: eigen-decompose \(A^\top A\) (or \(AA^\top\) if it's smaller) → \(\sigma=\sqrt\lambda\), \(V\) from eigenvectors, \(u_i=\frac1{\sigma_i}Av_i\). Verify columns are orthonormal.
  7. Diagonalizability: distinct eigenvalues ⇒ yes; repeated ⇒ check geometric vs algebraic multiplicity; symmetric ⇒ always (\(Q\Lambda Q^\top\)).
  8. Gradient identities save time: \(\nabla\|Ax-b\|^2=2A^\top(Ax-b)\), \(\nabla(x^\top Ax)=2Ax\) (symmetric).
  9. Chain rule: one term per intermediate variable / per path; write the tree before substituting.
  10. Classify critical points: \(\nabla f=0\) → Hessian → sign of \(D=f_{xx}f_{yy}-f_{xy}^2\) and \(f_{xx}\). \(D=0\) ⇒ say "inconclusive".
  11. Taylor: compute \(f(a),f'(a),f''(a),\dots\) and plug into \(\sum\frac{f^{(k)}(a)}{k!}(x-a)^k\). For composites, substitute known series and truncate.
  12. Verify wherever cheap: plug the solution back into \(Ax=b\); check \(QΛQ^\top=A\); confirm \(\langle x,x\rangle>0\). Visible verification earns marks.
