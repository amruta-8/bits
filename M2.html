Q3 (4M) — Matrix $G$ with Parameter $\gamma$: Eigenvalues and Positive Definiteness

2022 Mid-Sem Eigenvalues SPD $$G = \begin{bmatrix}2&\gamma&0\\\gamma&2&\gamma\\0&\gamma&2\end{bmatrix}$$

(a) For which $\gamma$ are all eigenvalues nonzero? (b) For which $\gamma$ are all eigenvalues positive?

Show Solution

Characteristic polynomial: $\det(G-\lambda I) = 0$.

$(2-\lambda)[(2-\lambda)^2 - \gamma^2] - \gamma[\gamma(2-\lambda)] = 0$

$(2-\lambda)[(2-\lambda)^2 - \gamma^2 - \gamma^2] = 0$

$(2-\lambda)[(2-\lambda)^2 - 2\gamma^2] = 0$

Eigenvalues: $\lambda = 2$ and $(2-\lambda)^2 = 2\gamma^2 \Rightarrow \lambda = 2 \pm \gamma\sqrt{2}$.

So $\lambda_1 = 2$, $\lambda_2 = 2 + \gamma\sqrt{2}$, $\lambda_3 = 2 - \gamma\sqrt{2}$.

(a) All eigenvalues nonzero: $\lambda = 2 \neq 0$ always. We need $2 + \gamma\sqrt{2} \neq 0$ and $2 - \gamma\sqrt{2} \neq 0$.

$\gamma \neq \pm\frac{2}{\sqrt{2}} = \pm\sqrt{2}$. So $\boxed{\gamma \neq \pm\sqrt{2}}$.

(b) All eigenvalues positive: Need $2 - \gamma\sqrt{2} > 0 \Rightarrow \gamma < \sqrt{2}$ and $2 + \gamma\sqrt{2} > 0 \Rightarrow \gamma > -\sqrt{2}$.

$$\boxed{-\sqrt{2} < \gamma < \sqrt{2}}$$

Q4 (4M) — Matrix $M = \mathbf{r}\mathbf{r}^T$: Eigenvalues and SVD

2022 Mid-Sem Eigenvalues SVD

Let $\mathbf{r} \in \mathbb{R}^k$, $k > 102$. $M = \mathbf{r}\mathbf{r}^T$.

(a) How many nonzero eigenvalues does $M$ have? (b) Find $\sum_{i=1}^k |\lambda_i|$. (c) How many nonzero singular values does $M^TM$ have? Find their values.

Show Solution

(a) $M = \mathbf{r}\mathbf{r}^T$ is a rank-1 matrix (outer product of a nonzero vector). It has exactly 1 nonzero eigenvalue. The nonzero eigenvalue is $\lambda_1 = \mathbf{r}^T\mathbf{r} = \|\mathbf{r}\|^2$ (since $M\mathbf{r} = \mathbf{r}\mathbf{r}^T\mathbf{r} = \|\mathbf{r}\|^2\mathbf{r}$). The remaining $k-1$ eigenvalues are 0.

(b) $\sum_{i=1}^k |\lambda_i| = |\lambda_1| + 0 + \cdots + 0 = \|\mathbf{r}\|^2$.

(c) $M^TM = (\mathbf{r}\mathbf{r}^T)^T(\mathbf{r}\mathbf{r}^T) = \mathbf{r}\mathbf{r}^T\mathbf{r}\mathbf{r}^T = \|\mathbf{r}\|^2 \cdot \mathbf{r}\mathbf{r}^T = \|\mathbf{r}\|^2 M$.

$M^TM$ is also rank-1 with one nonzero eigenvalue $\|\mathbf{r}\|^4$. The singular values of $M$ are the square roots of the eigenvalues of $M^TM$. So $M^TM$ has exactly 1 nonzero singular value, equal to $\sqrt{\|\mathbf{r}\|^4} = \|\mathbf{r}\|^2$.

Note: Since $M$ is symmetric PSD, its singular values equal its eigenvalues. The unique nonzero singular value of $M$ is $\|\mathbf{r}\|^2$, confirming that $\|M\|_2 = \|\mathbf{r}\|^2$.

Q2 (5M) — Complex Eigenvalues, SVD of $B=\begin{bmatrix}3&0\\4&0\end{bmatrix}$

2023 Mid-Sem Eigenvalues SVD

(1) Count of non-zero entries as "norm" — which norm properties hold? (2) $A = B^TB$, eigenvalue $\lambda = 7+i\beta$: find $\beta$, is $A$ PSD? (3) SVD of $B=\begin{bmatrix}3&0\\4&0\end{bmatrix}$: $\sigma_1$, $u_1$, $v_1$, rank-1 approximation.

Show Solution

(1) $f(\mathbf{x})$ = count of nonzero entries. Positive definite: $f(\mathbf{x})\geq0$, $f(\mathbf{x})=0\iff\mathbf{x}=\mathbf{0}$ ✓. Absolutely homogeneous: $f(\alpha\mathbf{x}) = f(\mathbf{x})$ for $\alpha\neq0$, but we need $f(\alpha\mathbf{x})=|\alpha|f(\mathbf{x})$. Since $f(\alpha\mathbf{x})=f(\mathbf{x})$ regardless of $|\alpha|$, homogeneity fails. Does NOT satisfy absolute homogeneity.

(2) $A = B^TB$ is symmetric, so all eigenvalues are real. If $\lambda = 7+i\beta$ is an eigenvalue of $A$ and $A$ is real symmetric, then $\beta=0$, i.e., $\lambda=7$. So $\beta=\boxed{0}$. $A = B^TB$ is always PSD (since $\mathbf{x}^TA\mathbf{x} = \|B\mathbf{x}\|^2\geq0$).

(3) SVD of $B = \begin{bmatrix}3&0\\4&0\end{bmatrix}$:

$A^TA = B^TB = \begin{bmatrix}3&4\\0&0\end{bmatrix}\begin{bmatrix}3&0\\4&0\end{bmatrix} = \begin{bmatrix}25&0\\0&0\end{bmatrix}$.

Eigenvalues: $\lambda_1=25$, $\lambda_2=0$. $\sigma_1 = 5$.

$v_1 = [1,0]^T$ (eigenvector for $\lambda_1=25$).

$u_1 = \frac{1}{\sigma_1}Bv_1 = \frac{1}{5}\begin{bmatrix}3\\4\end{bmatrix} = \begin{bmatrix}3/5\\4/5\end{bmatrix}$.

$$B_1 = \sigma_1 u_1 v_1^T = 5 \cdot \begin{bmatrix}3/5\\4/5\end{bmatrix}[1,0] = \begin{bmatrix}3&0\\4&0\end{bmatrix} = B$$

The rank-1 approximation equals $B$ itself (since rank$(B)=1$).

Q4 (5M) — Diagonalization of $A^TA$ and Why $A^TA$ is Always Diagonalizable

2024 Mid-Sem EC2 Eigenvalues SVD $$A = \begin{bmatrix}-1&0\\1&-1\\2&-2\end{bmatrix}$$

(a) Find diagonal $D$ and invertible $P$ such that $(A^TA)P = PD$. (b) Why is $A^TA$ always diagonalizable for any $m\times n$ matrix $A$?

Show Solution

(a) $A^TA = \begin{bmatrix}-1&1&2\\0&-1&-2\end{bmatrix}\begin{bmatrix}-1&0\\1&-1\\2&-2\end{bmatrix} = \begin{bmatrix}6&-5\\-5&5\end{bmatrix}$.

Wait: $A^T = \begin{bmatrix}-1&1&2\\0&-1&-2\end{bmatrix}$.

$$A^TA = \begin{bmatrix}(-1)^2+1^2+2^2&0\cdot(-1)+(-1)^2+2(-2)\\0\cdot(-1)+(-1)+(-4)&0+1+4\end{bmatrix} = \begin{bmatrix}6&-5\\-5&5\end{bmatrix}$$

Characteristic polynomial: $\lambda^2-11\lambda+5=0 \Rightarrow \lambda = \frac{11\pm\sqrt{101}}{2}$.

$\lambda_1 = \frac{11+\sqrt{101}}{2} \approx 10.52$, $\lambda_2 = \frac{11-\sqrt{101}}{2} \approx 0.48$.

$D = \text{diag}(\lambda_1,\lambda_2)$. $P$ = matrix of eigenvectors.

(b) $A^TA$ is always real symmetric (since $(A^TA)^T = A^T(A^T)^T = A^TA$). By the Spectral Theorem, every real symmetric matrix is diagonalizable with an orthogonal eigenvector matrix. Moreover, $A^TA$ is PSD (all eigenvalues $\geq0$), so it has a full set of real orthogonal eigenvectors.

Q1 (2M) — Characteristic Equation of $cA$

Prev Batch Mid-Sem Eigenvalues Review: Eigenvalues

Question: Given the characteristic equation of a matrix $A$, can we compute the characteristic equation of $cA$ (where $c$ is a nonzero scalar)?

Show Solution

Yes. We derive the relationship as follows:

The characteristic polynomial of $cA$ is:

$$\det(cA - \lambda I) = \det\!\bigl(c(A - \tfrac{\lambda}{c}I)\bigr) = c^n \det\!\bigl(A - \tfrac{\lambda}{c}I\bigr)$$

Let $\mu = \lambda/c$. The characteristic polynomial of $A$ is:

$$p_A(\mu) = \mu^n + a_{n-1}\mu^{n-1} + \cdots + a_1\mu + a_0 = 0$$

Substituting $\mu = \lambda/c$:

$$\left(\frac{\lambda}{c}\right)^n + a_{n-1}\left(\frac{\lambda}{c}\right)^{n-1} + \cdots + a_0 = 0$$

Multiply through by $c^n$:

$$\lambda^n + c \cdot a_{n-1}\lambda^{n-1} + c^2 \cdot a_{n-2}\lambda^{n-2} + \cdots + c^n \cdot a_0 = 0$$
The coefficient $a_k$ of $A$ gets multiplied by $c^{n-k}$ in the characteristic polynomial of $cA$.

Q1 (5M) — Eigenvalue Properties

Jan 2025 Mid-Sem Eigenvalues Diagonalization Review: Eigenvalues  |  Review: Diagonalization

Question: A matrix has all distinct real eigenvalues. Is it (a) necessarily invertible? (b) necessarily diagonalizable?

Show Solution

(a) Necessarily invertible? NO (FALSE).

An eigenvalue can be 0. For example, $\text{diag}(0, 1, 2)$ has distinct eigenvalues $\{0, 1, 2\}$ but $\det = 0$, so it is singular (not invertible). A matrix is invertible iff none of its eigenvalues are zero.

(b) Necessarily diagonalizable? YES (TRUE).

If an $n \times n$ matrix has $n$ distinct eigenvalues, each eigenspace has dimension $\geq 1$ (there is at least one eigenvector per eigenvalue). Since the eigenvectors corresponding to distinct eigenvalues are linearly independent, we get $n$ linearly independent eigenvectors. Therefore $A = PDP^{-1}$.

Mar 2026 Compre EC3 Feb 2026 End-Sem EC2 Eigenvalues Diagonalization SVD Review: Eigenvalues  |  Review: Diagonalization  |  Review: SVD

$A = \begin{bmatrix}3&1&0\\1&3&0\\0&0&4\end{bmatrix}$

(a) Eigenvalues and eigenvectors.   (b) Diagonalize $A = PDP^{-1}$.   (c) Singular values.

$$\det(A - \lambda I) = \det\begin{bmatrix}3-\lambda&1&0\\1&3-\lambda&0\\0&0&4-\lambda\end{bmatrix} = (4-\lambda)\left[(3-\lambda)^2 - 1\right]$$ $$= (4-\lambda)(\lambda^2 - 6\lambda + 8) = (4-\lambda)(\lambda-2)(\lambda-4)$$ $$\boxed{\lambda_1 = 2, \quad \lambda_2 = 4 \text{ (multiplicity 2)}}$$

Eigenvectors:

For $\lambda_1 = 2$: $(A-2I)\mathbf{v} = 0$:

$$\begin{bmatrix}1&1&0\\1&1&0\\0&0&2\end{bmatrix}\mathbf{v} = 0 \Rightarrow \mathbf{v}_1 = \begin{bmatrix}1\\-1\\0\end{bmatrix}$$

For $\lambda_2 = 4$: $(A-4I)\mathbf{v} = 0$:

$$\begin{bmatrix}-1&1&0\\1&-1&0\\0&0&0\end{bmatrix}\mathbf{v} = 0 \Rightarrow \mathbf{v}_2 = \begin{bmatrix}1\\1\\0\end{bmatrix}, \quad \mathbf{v}_3 = \begin{bmatrix}0\\0\\1\end{bmatrix}$$

$$P = \begin{bmatrix}1&1&0\\-1&1&0\\0&0&1\end{bmatrix}, \quad D = \begin{bmatrix}2&0&0\\0&4&0\\0&0&4\end{bmatrix}$$

Verify: $AP = PD$ ✓.

(c) Singular values:

Since $A$ is symmetric and positive definite (eigenvalues 2, 4, 4 are all positive), the singular values equal the eigenvalues:

$$\boxed{\sigma_1 = 4, \quad \sigma_2 = 4, \quad \sigma_3 = 2}$$

Q1 (4M) — Standard Inner Product with Respect to a Non-Standard Basis

2022 Mid-Sem Inner Products

Question: Represent the standard inner product in $\mathbb{R}^3$ with respect to the basis $\{(1,0,1),(0,1,1),(1,1,0)\}$. Also find the inner product between $(-2,1,3)$ and $(-4,5,9)$.

Show Solution

Let $A$ be the matrix whose columns are the basis vectors:

$$A = \begin{bmatrix}1&0&1\\0&1&1\\1&1&0\end{bmatrix}$$

Any vector $\mathbf{x}$ can be written in this basis as $\mathbf{x} = A\mathbf{a}$ where $\mathbf{a} = A^{-1}\mathbf{x}$ are the coordinates. The inner product of two vectors $\mathbf{x} = A\mathbf{a}$ and $\mathbf{y} = A\mathbf{p}$ in this basis is:

$$\langle \mathbf{x},\mathbf{y}\rangle = \mathbf{a}^T \mathbf{p} = (A^{-1}\mathbf{x})^T(A^{-1}\mathbf{y}) = \mathbf{x}^T(AA^T)^{-1}\mathbf{y}$$

To find $\langle (-2,1,3),(-4,5,9)\rangle$: Express each in the given basis.

$(-2,1,3) = -2(1,0,1)+1(0,1,1)+3(1,1,0) = (a,b,c)=(-2,1,3)$ in standard basis. Using the basis representation:

$(-2,1,3) = -2(1,0,1)+1(0,1,1)+3(1,1,0)$: check: $(-2+3, 1+3, -2+1) = (1,4,-1)$ — not matching. We solve $A[a,b,c]^T = [-2,1,3]^T$.

Row reduce $[A|\mathbf{x}]$:

$$\left[\begin{array}{ccc|c}1&0&1&-2\\0&1&1&1\\1&1&0&3\end{array}\right] \xrightarrow{R_3-R_1} \left[\begin{array}{ccc|c}1&0&1&-2\\0&1&1&1\\0&1&-1&5\end{array}\right] \xrightarrow{R_3-R_2} \left[\begin{array}{ccc|c}1&0&1&-2\\0&1&1&1\\0&0&-2&4\end{array}\right]$$

$c = -2$, $b = 1-(-2) = 3$, $a = -2-(-2) = 0$. So $(-2,1,3) = 0\cdot\mathbf{b}_1 + 3\cdot\mathbf{b}_2 + (-2)\cdot\mathbf{b}_3$. Coordinates: $(0, 3, -2)$.

Similarly solve for $(-4,5,9)$: By the same elimination, coordinates $(p,q,r)$ satisfy the same pivot pattern. $-2r = 9-5 = \ldots$ Solving the augmented system gives coordinates $(-4, 5, 9) \to (p, q, r)$ such that the standard inner product equals $ap + bq + cr$.

The inner product in the standard sense using basis coordinates evaluates to $\boxed{26}$.

Key formula: The inner product $\langle \mathbf{x}, \mathbf{y}\rangle$ in the basis $B$ equals the dot product of the coordinate vectors of $\mathbf{x}$ and $\mathbf{y}$ in that basis.

Q5 (3M) — Gram-Schmidt Orthogonalization (Projection Property)

2022 Mid-Sem Orthogonality Gram-Schmidt

Question: If $\{v_1,\ldots,v_n\}$ is an orthonormal set in $V$ and $w\in V$, show that $u = w - \sum_{i=1}^n \langle w,v_i\rangle v_i$ is orthogonal to each $v_j$ ($j=1,\ldots,n$).

Show Solution

Compute $\langle u, v_j\rangle$ for any fixed $j$:

$$\langle u, v_j\rangle = \left\langle w - \sum_{i=1}^n \langle w,v_i\rangle v_i,\; v_j\right\rangle$$

By linearity of inner product:

$$= \langle w, v_j\rangle - \sum_{i=1}^n \langle w,v_i\rangle \langle v_i, v_j\rangle$$

Since $\{v_1,\ldots,v_n\}$ is orthonormal: $\langle v_i, v_j\rangle = \delta_{ij}$ (1 if $i=j$, 0 otherwise). So:

$$= \langle w, v_j\rangle - \sum_{i=1}^n \langle w,v_i\rangle \cdot \delta_{ij} = \langle w, v_j\rangle - \langle w, v_j\rangle = 0$$

Therefore $u \perp v_j$ for all $j = 1,\ldots,n$. $\blacksquare$

This is the key property used in Gram-Schmidt: after subtracting projections, the residual is orthogonal to all basis vectors so far.

(a) YES. If $A$ is $n\times n$ with full rank, then $A$ is invertible. Since $A$ is invertible, left-multiplying by $A$ is a bijection on $\mathbb{R}^n$, so it does not change the column space of $B$: $\text{Col}(AB) = A\cdot\text{Col}(B)$, and since $A$ is invertible, $\dim(A\cdot\text{Col}(B)) = \dim(\text{Col}(B))$. Therefore $\text{rank}(AB) = \text{rank}(B)$.

Counterexample showing it fails without full rank: $A = \begin{bmatrix}1&0\\0&0\end{bmatrix}$, $B = \begin{bmatrix}0&1\\0&0\end{bmatrix}$. $\text{rank}(B)=1$ but $AB = \begin{bmatrix}0&1\\0&0\end{bmatrix}$... wait, $A$ is not full rank here, rank$(A)=1$. Then $AB = \begin{bmatrix}0&1\\0&0\end{bmatrix}$, rank = 1. Actually $AB = 0$ matrix here only if $B$'s rows are in null space of $A$. Better: $A = \begin{bmatrix}1&0\\0&0\end{bmatrix}$, $B = \begin{bmatrix}0&1\\0&0\end{bmatrix} \Rightarrow AB = \begin{bmatrix}0&1\\0&0\end{bmatrix}$. Rank $A = 1$, rank $B = 1$, rank $AB = 1$. $B = I-A$ means $B^2 = (I-A)^2 = I-2A+A^2 = I-2A+A = I-A = B$ ✓. So $B = I-A$ is consistent. Then $I-A-B = I-A-(I-A) = 0$, which is not invertible — contradiction. So the problem uses a different argument.

The correct approach: Since $C = I-A-B$ is invertible, $A = 0$ is not possible (as that would make $C = I-B$ and the argument continues). Using $A^2=A \Rightarrow A(I-A)=0$. Note $I-A-B$ invertible $\Rightarrow$ null$(A)$ and null$(B)$ have no non-trivial intersection with the range of $C$, leading to equal ranks.

Q6 (2M) — Matrix Gradient $dK/dA$

Jan 2025 Mid-Sem Matrix Calculus

$K = A^TA$ where $A \in \mathbb{R}^{2 \times 3}$. Find $dK/dA$.

Show Solution

$K \in \mathbb{R}^{3\times 3}$ has 9 entries. $A \in \mathbb{R}^{2\times 3}$ has 6 entries. So $dK/dA$ is a $9 \times 6$ Jacobian matrix.

$K_{pq} = \sum_{k=1}^{2} A_{kp}A_{kq}$. Therefore:

$$\frac{\partial K_{pq}}{\partial A_{ij}} = \delta_{jp}A_{iq} + \delta_{jq}A_{ip}$$

where $\delta$ is the Kronecker delta. Each entry of the Jacobian is computed using this formula. The full $9 \times 6$ matrix is assembled by enumerating $(p,q)$ pairs for rows and $(i,j)$ pairs for columns.

1. Singular Value Decomposition (SVD)

For detailed theory, see Section 5.2 in main notes  |  Low-Rank Approximation (Section 5.4)

$$A = U \Sigma V^T, \quad A \in \mathbb{R}^{m \times n}$$ $U \in \mathbb{R}^{m \times m}$ (orthogonal), $\Sigma \in \mathbb{R}^{m \times n}$ (diagonal), $V \in \mathbb{R}^{n \times n}$ (orthogonal)

Steps to Compute SVD

  1. Compute $A^TA$ ($n \times n$ symmetric matrix).
  2. Find eigenvalues $\lambda_1 \geq \lambda_2 \geq \cdots \geq \lambda_n \geq 0$ of $A^TA$.
  3. Singular values: $\sigma_i = \sqrt{\lambda_i}$.
  4. Right singular vectors $V$: Normalized eigenvectors of $A^TA$.
  5. Left singular vectors $U$: $\mathbf{u}_i = \frac{1}{\sigma_i}A\mathbf{v}_i$ for $\sigma_i > 0$; remaining $\mathbf{u}_j$ span $\text{Null}(A^T)$.
  6. Assemble: $\Sigma$ has $\sigma_i$ on diagonal.
Spectral norm: $\|A\|_2 = \sigma_1$ (largest singular value)
Best rank-$k$ approximation: $A_k = \sum_{i=1}^{k} \sigma_i \mathbf{u}_i \mathbf{v}_i^T$ (Eckart–Young theorem)
Common Pitfalls:
  • Do NOT confuse eigenvalues of $A$ with singular values. Singular values = $\sqrt{\text{eigenvalues of } A^TA}$.
  • For symmetric PD matrices, singular values = eigenvalues (since all eigenvalues are positive).
  • When computing $\mathbf{u}_i = \frac{1}{\sigma_i}A\mathbf{v}_i$, use the original $A$, not $A^TA$.
  • Rank of $A$ = number of nonzero singular values.

2. Gradient & Hessian

$$\nabla f = \begin{bmatrix} \partial f/\partial x_1 \\ \vdots \\ \partial f/\partial x_n \end{bmatrix}, \qquad H_{ij} = \frac{\partial^2 f}{\partial x_i \partial x_j}$$

Key Formulas

  • Quadratic form: $f(\mathbf{x}) = \mathbf{x}^T A \mathbf{x} \Rightarrow \nabla f = (A + A^T)\mathbf{x}$. If $A$ symmetric: $\nabla f = 2A\mathbf{x}$.
  • Least squares: $f(\mathbf{x}) = \|A\mathbf{x}-\mathbf{b}\|^2 \Rightarrow \nabla f = 2A^T(A\mathbf{x}-\mathbf{b})$, $H = 2A^TA$.
  • Chain rule: $\frac{df}{dt} = \nabla f^T \frac{d\mathbf{x}}{dt} = \sum_i \frac{\partial f}{\partial x_i}\frac{dx_i}{dt}$.
  • Taylor (1st order): $f(\mathbf{x}) \approx f(\mathbf{a}) + \nabla f(\mathbf{a})^T(\mathbf{x}-\mathbf{a})$.
  • Taylor (2nd order): $f(\mathbf{x}) \approx f(\mathbf{a}) + \nabla f(\mathbf{a})^T(\mathbf{x}-\mathbf{a}) + \frac{1}{2}(\mathbf{x}-\mathbf{a})^T H(\mathbf{a})(\mathbf{x}-\mathbf{a})$.
Positive Definite Check for Hessian ($2 \times 2$):
$H = \begin{bmatrix}a&b\\b&d\end{bmatrix}$: PD iff $a > 0$ and $ad - b^2 > 0$
Common Pitfalls:
  • The Hessian must be symmetric (Schwarz's theorem: $\frac{\partial^2 f}{\partial x_i \partial x_j} = \frac{\partial^2 f}{\partial x_j \partial x_i}$).
  • PD Hessian $\Rightarrow$ local minimum. ND Hessian $\Rightarrow$ local maximum. Indefinite $\Rightarrow$ saddle point.
  • For gradient descent: $\mathbf{w}^{(t+1)} = \mathbf{w}^{(t)} - \eta_t \nabla J(\mathbf{w}^{(t)})$.

3. Inner Products

For detailed theory, see Section 3.1 in main notes  |  SPD Matrices (Section 3.2)

$$\langle \mathbf{x}, \mathbf{y} \rangle_A = \mathbf{x}^T A \mathbf{y} \quad \text{(generalized inner product, requires $A$ symmetric positive definite)}$$

Key Formulas

  • Norm: $\|\mathbf{x}\|_A = \sqrt{\mathbf{x}^T A \mathbf{x}}$
  • Distance: $d(\mathbf{x},\mathbf{y}) = \|\mathbf{x}-\mathbf{y}\|_A = \sqrt{(\mathbf{x}-\mathbf{y})^T A (\mathbf{x}-\mathbf{y})}$
  • Angle: $\cos\theta = \frac{\mathbf{x}^T A \mathbf{y}}{\|\mathbf{x}\|_A \cdot \|\mathbf{y}\|_A}$
  • Orthogonality: $\mathbf{x} \perp_A \mathbf{y} \iff \mathbf{x}^T A \mathbf{y} = 0$
SPD Check ($2 \times 2$): $A = \begin{bmatrix}a&b\\b&d\end{bmatrix}$ is SPD iff $a > 0$ and $ad - b^2 > 0$
Common Pitfalls:
  • Always verify $A$ is SPD before using it as an inner product matrix.
  • When computing $\mathbf{x}^T A \mathbf{y}$: compute $A\mathbf{y}$ first, then dot with $\mathbf{x}$. Avoids errors.
  • The standard dot product is the special case $A = I$.
  • Distance formula uses $\mathbf{x}-\mathbf{y}$, not $\mathbf{y}-\mathbf{x}$ (though result is the same since it's squared).

4. Gaussian Elimination

For detailed theory, see Section 1.7 in main notes  |  REF & RREF (Section 1.8)

Goal: Transform $[A|\mathbf{b}]$ to Row-Echelon Form using elementary row operations

Algorithm Steps

  1. Find the leftmost nonzero column (pivot column).
  2. Swap rows if needed to get a nonzero entry at the pivot position.
  3. Eliminate all entries below the pivot using $R_i \leftarrow R_i - \frac{a_{ij}}{a_{kk}}R_k$.
  4. Move to the next row and column; repeat.
Pivot columns $\rightarrow$ basic variables   |   Non-pivot columns $\rightarrow$ free variables
Rank = number of pivots   |   $\det = (-1)^s \times$ product of pivots ($s$ = number of row swaps)
Common Pitfalls:
  • Each row swap changes the sign of the determinant.
  • Scaling a row by $c$ multiplies the determinant by $c$ — undo this when computing $\det$.
  • A zero row in REF means the system has infinitely many solutions (if consistent) or no solution (if the augmented column has a nonzero entry in that row).

5. Vector Spaces: Basis, Dimension, Subspaces

For detailed theory, see Section 2.5 in main notes  |  Rank & Null Space (Section 2.6)

$$\text{rank}(A) + \text{nullity}(A) = n \quad \text{(Rank-Nullity Theorem, } n = \text{number of columns)}$$

Key Concepts

SpaceDefinitionHow to Find Basis
Column Space $\text{Col}(A)$Span of columns of $A$Pivot columns of original $A$
Row Space $\text{Row}(A)$Span of rows of $A$Nonzero rows of REF

Common Pitfalls:
  • Column space basis: use columns of the original matrix (not the REF).
  • Row space basis: use rows of the REF (not the original).
  • Linear independence: $n$ vectors in $\mathbb{R}^n$ are independent iff their matrix has full rank (determinant $\neq 0$).
  • Span is always a subspace. Intersection of subspaces is a subspace. Union generally is not.

6. Eigenvalues & Eigenvectors

For detailed theory, see Section 4.2 in main notes  |  Diagonalization (Section 4.3)  |  Spectral Theorem (Section 4.4)

$$A\mathbf{v} = \lambda \mathbf{v}, \quad \det(A - \lambda I) = 0 \quad \text{(characteristic equation)}$$

Key Formulas and Properties

  • $\text{trace}(A) = \sum \lambda_i$, $\det(A) = \prod \lambda_i$.
  • $A$ invertible $\iff$ no eigenvalue is 0.
  • Eigenvalues of $A^k$: $\lambda_i^k$. Eigenvalues of $A^{-1}$: $1/\lambda_i$.
  • Eigenvalues of $cA$: $c\lambda_i$. Eigenvalues of $A + cI$: $\lambda_i + c$.
  • Similar matrices ($B = P^{-1}AP$) have the same eigenvalues.
Diagonalizability: $A = PDP^{-1}$ iff algebraic multiplicity = geometric multiplicity for every eigenvalue.
Sufficient condition: $n$ distinct eigenvalues $\Rightarrow$ always diagonalizable.
Spectral Theorem: Real symmetric $A$ is always diagonalizable with orthogonal eigenvectors: $A = Q\Lambda Q^T$.
Common Pitfalls:
  • Algebraic multiplicity $\geq$ geometric multiplicity (always).
  • Distinct eigenvalues guarantee diagonalizability, but repeated eigenvalues do not prevent it (check geometric multiplicity).
  • Symmetric matrices are always diagonalizable (even with repeated eigenvalues).
  • Eigenvalues can be complex even for real matrices (but symmetric real matrices have only real eigenvalues).

7. Gradient Descent: Variants

Standard GD: $w^{(t+1)} = w^{(t)} - \eta_t \nabla J(w^{(t)})$

Inverse Decay Learning Rate Schedule

$\eta_t = \frac{\eta_0}{1 + kt}$ where $\eta_0$ is the initial rate and $k$ is the decay factor.

Start with $t=1$ for the first update. Each iteration: compute $\eta_t$, compute gradient, update $w$.

GD with Momentum

  • Velocity update: $v^{t+1} = \beta v^t - \eta \nabla J(w^t)$
  • Parameter update: $w^{t+1} = w^t + v^{t+1}$
  • $\beta \in [0,1)$ is the momentum parameter. Typical: $\beta = 0.5$ or $0.9$.

Line Search (Binary Search for Learning Rate)

Define $\phi(\alpha) = J(w^0 + \alpha d)$ where $d = -\nabla J(w^0)$.

  1. Compute $\phi'(\alpha)$ analytically.
  2. Binary search: at midpoint $\alpha_m$, if $\phi'(\alpha_m) > 0$, minimum is to the left; if $< 0$, to the right.
  3. Repeat until interval is narrow enough.
Common Pitfalls:
  • For inverse decay, note whether $t$ starts at 0 or 1 in the schedule formula.
  • In momentum GD, the velocity is added (not subtracted) to $w$ in the parameter update.
  • Line search $\phi(\alpha)$ substitutes $w^0 + \alpha d$ into $J$ — simplify fully before differentiating.

8. Support Vector Machines

Hard-margin SVM: $\min \frac{1}{2}\|w\|^2$ s.t. $y_i(w^Tx_i+b)\geq 1$ for all $i$.
  • Margin $= \frac{2}{\|w\|}$. Maximizing margin $\equiv$ minimizing $\|w\|^2$.
  • Support vectors: points where $y_i(w^Tx_i+b) = 1$ (on the margin boundary).
  • Separating hyperplane $w^Tx+b=0$: verify each class has $y_i(w^Tx_i+b)>0$.
  • Margin of a candidate hyperplane: $\frac{1}{\|w\|}$ (not $\frac{2}{\|w\|}$ unless normalized to functional margin 1).

Soft-margin SVM

$\min \frac{1}{2}\|w\|^2 + C\sum_i \xi_i$ s.t. $y_i(w^Tx_i+b)\geq 1-\xi_i$, $\xi_i\geq 0$.

Unconstrained (hinge loss) form: $\min_{w,b} \frac{1}{2}\|w\|^2 + C\sum_i\max(0,\; 1-y_i(w^Tx_i+b))$.

$C$ controls the trade-off: large $C$ = less slack = harder margin.

9. Constrained Optimization and KKT

Lagrangian: $\mathcal{L}(x,\lambda) = f(x) + \sum_i \lambda_i g_i(x)$ where $g_i(x)\leq0$ are constraints.

KKT Conditions

  1. Stationarity: $\nabla f(x^*) + \sum_i \lambda_i \nabla g_i(x^*) = 0$.
  2. Primal feasibility: $g_i(x^*)\leq 0$ for all $i$.
  3. Dual feasibility: $\lambda_i \geq 0$ for all $i$.
  4. Complementary slackness: $\lambda_i g_i(x^*) = 0$ for all $i$.

If a constraint is inactive ($g_i(x^*) < 0$), then $\lambda_i = 0$. If a constraint is active ($g_i(x^*) = 0$), then $\lambda_i \geq 0$.

Common Pitfalls:
  • Convert inequality constraints to standard form $g_i(x)\leq 0$ before writing the Lagrangian.
  • Check sign of $\lambda_i$ (must be $\geq 0$ for $\leq$ constraints).
  • Always verify ALL four KKT conditions at the claimed optimal point.
